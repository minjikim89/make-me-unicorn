# Brand Assets

Place your unicorn image files here.

## Expected Files

- `unicorn-hero.png` (README hero image)
- `og-cover.png` (social preview image, required 1200x630)

## Usage

- README hero references: `assets/brand/unicorn-hero.png`
- Social preview uses: `assets/brand/og-cover.png`

## Notes

- Keep source files in PNG format for predictable rendering.
- Re-export optimized variants if file size is above 1 MB.
