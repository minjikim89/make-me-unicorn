# SEO and Distribution Checklist

## Metadata

- [ ] `title` and `description` are set per page.
- [ ] Canonical URL is set.
- [ ] Open Graph tags are valid (including `og:locale`).
- [ ] Twitter Card tags are valid.
- [ ] `hreflang` tags are set (if multi-language).

## Thumbnail and Brand Surface

- [ ] OG thumbnail (1200x630) exists.
- [ ] Favicon/app icon exists.
- [ ] Shared preview title/description matches product messaging.
- [ ] Link previews are verified on real channels (Slack/Kakao/X/Messenger).

## Indexing and Search Registration

- [ ] `sitemap.xml` is generated and accessible.
- [ ] `robots.txt` is configured.
- [ ] Noindex pages are intentionally scoped.
- [ ] Google Search Console registered and domain verified.
- [ ] Sitemap submitted in Google Search Console.
- [ ] Key pages indexed (URL Inspection â†’ Request Indexing).
- [ ] Bing Webmaster Tools registered (if applicable).
- [ ] Naver Search Advisor registered (if targeting Korea).

## Analytics Baseline

- [ ] GA4 installed and receiving data.
- [ ] GA4 linked to Google Search Console.
- [ ] Core conversion events defined and tracked.

## Content and Growth

- [ ] Landing page has clear primary keyword intent.
- [ ] Comparison/use-case content plan exists.
- [ ] UTM and attribution policy is documented.

## Post-Launch Verification

- [ ] Google Rich Results Test passes for key pages.
- [ ] Mobile-Friendly Test passes.
- [ ] Core Web Vitals pass in PageSpeed Insights.
- [ ] Redirects (www/non-www, trailing slash) work correctly.
