# SEO and Distribution Checklist

## Metadata

- [ ] `title` and `description` are set per page.
- [ ] Canonical URL is set.
- [ ] Open Graph tags are valid.
- [ ] Twitter Card tags are valid.

## Thumbnail and Brand Surface

- [ ] OG thumbnail (1200x630) exists.
- [ ] Favicon/app icon exists.
- [ ] Shared preview title/description matches product messaging.
- [ ] Link previews are verified on real channels (Slack/Kakao/X/Messenger).

## Indexing

- [ ] `sitemap.xml` is generated.
- [ ] `robots.txt` is configured.
- [ ] Noindex pages are intentionally scoped.

## Content and Growth

- [ ] Landing page has clear primary keyword intent.
- [ ] Comparison/use-case content plan exists.
- [ ] UTM and attribution policy is documented.
