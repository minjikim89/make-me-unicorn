# Release Readiness Checklist

## Product

- [ ] Core user flow works for real usage.
- [ ] Error/empty/loading states are validated.
- [ ] Desktop and mobile behavior meet baseline quality.

## Engineering

- [ ] Environment variables and secrets are managed.
- [ ] Migration and rollback plans exist.
- [ ] Critical alerts are connected.

## Security and Compliance

- [ ] Auth/security checklist is passed.
- [ ] Policy pages and support contact path are live.
- [ ] Data deletion/change request workflow is defined.

## Billing

- [ ] Payment success/failure/refund tests are passed.
- [ ] Webhook failure retry strategy exists.
- [ ] Plan-based access control is verified.

## Growth and Analytics

- [ ] OG/SEO metadata is validated.
- [ ] Core events are visible in dashboard.
- [ ] Launch copy and onboarding notes are ready.
