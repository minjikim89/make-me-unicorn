# Industry Blueprints

Additional checklists for specific product types. Use these **alongside** the core 15 blueprints.

| Blueprint | Items | For products that... |
|-----------|-------|---------------------|
| [AI Product](./ai-product.md) | 45+ | Use LLMs, ML models, or AI APIs as core features |
| [Marketplace](./marketplace.md) | 55+ | Connect buyers and sellers in a two-sided market |

## How to Use

1. Complete the core 15 blueprints first (they cover universal SaaS needs).
2. Pick the industry blueprint that matches your product type.
3. Work through the additional items — they catch domain-specific blind spots.
