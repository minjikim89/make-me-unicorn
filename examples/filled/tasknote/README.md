# TaskNote (Filled Example)

This directory contains a filled-out example of how to operate Make Me Unicorn for a real SaaS scenario.

## Product

TaskNote is a lightweight AI-assisted task tracker for solo builders.

## Why this exists

The root templates are intentionally generic. This example shows concrete writing style,
scope decisions, and gate progression from M0 to M2.

## Included

- Core docs (`strategy`, `product`, `pricing`, `architecture`, `ux`)
- Ops docs (`roadmap`, `metrics`, `compliance`)
- One ADR (`billing provider decision`)
- A partially completed from-scratch checklist
- A realistic current sprint board
