# Compliance

## Legal Baseline

- Terms of service page: drafted
- Privacy policy page: published
- Cookie/tracking disclosure: published in footer

## Data Governance

- Data collected: email, project metadata, usage events
- Retention policy: account data retained until deletion request
- Deletion request process: in-app request + email confirmation
- Export request process: CSV export within 7 days

## Billing and Tax

- Tax/VAT handling: delegated via Stripe Tax
- Invoice/receipt flow: automatic email receipts
- Refund and dispute process: support ticket with decision SLA 3 days

## Security and Access

- Admin access controls: limited to break-glass accounts
- Secret management policy: environment secrets only
- Audit logging policy: security events retained 90 days

## Region Notes

- Countries/regions served: US, EU, KR
- Region-specific constraints: GDPR deletion/export handling required
