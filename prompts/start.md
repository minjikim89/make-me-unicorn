# Session Start Prompt

Run this workflow for the current session.

1. Confirm the selected `mode`.
2. Read only mode-relevant documents.
3. Re-prioritize up to 3 goals from `current_sprint.md`.
4. Separate "decision items" from "execution items".
5. Pre-declare which documents must be updated at session close.

Output format:

- Mode:
- Today goals (max 3):
- Decision items:
- Execution items:
- End-of-session update targets:
